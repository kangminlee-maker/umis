# Cursor Composer 상세 워크플로우

**대상:** UMIS 사용자 (코딩 불필요)  
**환경:** Cursor Composer + .cursorrules 자동화

---

## 🎯 전체 프로세스

### Phase 1: 분석 시작

```
Cursor Composer (Cmd+I):

You: "피아노 학원 구독 서비스로 전환하는 시장 기회 분석
     
     @umis.yaml"

AI: 
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🚀 UMIS 시장 분석 시작
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  [Observer (Observer) 작업]
  
  시장 관찰:
  - 피아노 구매: 300-800만원 (높은 진입장벽)
  - 정기 사용: 매일 연습 필요
  - 유지관리: 조율 연 1-2회, 15만원
  
  트리거 시그널 발견:
  ✅ 높은 초기 구매 비용
  ✅ 정기적 사용 필요
  ✅ 유지관리 필요
  ✅ 재구매율 낮음
  
  → Explorer에게 전달
```

### Phase 2: 패턴 매칭 (자동 RAG!)

```
AI:
  [Explorer (Explorer) 작업]
  
  패턴 매칭이 필요합니다. RAG에서 검색하겠습니다.
  
  [Agent 자동 실행]
  🔍 RAG 검색 중...
  
  python scripts/query_rag.py pattern "높은 초기 비용, 정기 사용, 유지관리"
  
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ 패턴 발견!
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  🥇 subscription_model (유사도: 1.03)
     - 본질: 소유 → 이용 전환
     - 핵심 가치: 초기 부담 감소, 안정 현금흐름
  
  → 이 패턴을 적용하겠습니다!
```

### Phase 3: 사례 검색 (자동!)

```
AI:
  유사 성공 사례를 검색하겠습니다.
  
  [Agent 자동 실행]
  🔍 사례 검색 중...
  
  python scripts/query_rag.py case "정수기 렌탈" --pattern subscription_model
  
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ 유사 사례 발견!
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  🥇 코웨이 (정수기 렌탈)
     - 구조: 100만원 구매 → 월 3만원
     - 규모: 655만 계정
     - 핵심: 정기 방문 = Lock-in
  
  🥈 멜론 (음악 스트리밍)
  
  → 코웨이가 피아노와 가장 유사합니다!
```

### Phase 4: 발견 & 개선

```
You: "코웨이 사례에 해지율 데이터가 없네. 추가해줘"

AI:
  알겠습니다. YAML 파일을 수정하겠습니다.
  
  [파일 열기]
  umis_business_model_patterns.yaml
  
  [위치 찾기]
  라인 348: 코웨이 섹션
  
  [수정 제안]
  ```diff
  코웨이:
    economics:
      arpu: "월 ~3만원"
      annual_revenue: "연 ~2.4조원"
  +   churn_rate: "3-5% (업계 평균)"
  ```
  
  이렇게 추가할까요?

You: "응"

AI:
  [파일 저장]
  ✅ 저장 완료!
  
  RAG 인덱스를 재구축할까요? (2초 소요)

You: "응"

AI:
  [Agent 자동 실행]
  🔄 RAG 재구축 중...
  
  python scripts/01_convert_yaml.py
  python scripts/02_build_index.py --agent explorer
  
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ RAG 업데이트 완료! (2초)
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  이제 Explorer가 코웨이 해지율 데이터를 사용할 수 있습니다.

You: "확인해봐"

AI:
  [Agent 실행]
  python scripts/query_rag.py case "코웨이"
  
  결과:
  🥇 코웨이
     churn_rate: "3-5%" ← 방금 추가한 데이터! ✅
  
  Explorer: "피아노도 유사 해지율 3-5% 예상됩니다..."
```

## 🎨 Agent 역할 (ID 기준)

```yaml
Observer (Observer):
  • 시장 구조 관찰
  • 트리거 시그널 발견
  
Explorer (Explorer):
  • 패턴 매칭 (RAG 활용!)
  • 기회 가설 생성
  • 사례 학습 (RAG 활용!)
  
Quantifier (Quantifier):
  • 시장 규모 계산
  • 수익성 분석
  
Validator (Validator):
  • 데이터 검증
  • 출처 확인
  
Guardian (Guardian):
  • 품질 관리
  • 순환 감지 (Memory-RAG!)
  • 목표 정렬 (Memory-RAG!)
```
