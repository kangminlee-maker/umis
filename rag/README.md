## 🤖 UMIS Agent (ID 기준)

```
Observer → 시장 관찰
Explorer → 기회 발굴 (RAG 활용!)
Quantifier → 정량 분석
Validator → 데이터 검증
Guardian → 프로세스 감시 (Memory-RAG!)
Owner → 의사결정
```
