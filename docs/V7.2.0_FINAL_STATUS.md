## 🔄 마이그레이션 가이드

### v7.1.0 → v7.2.0

**필수 작업:** 없음 (완전 하위 호환)

**선택적 개선:**

#### 1. 환경변수 로드 간소화
```python
# Before (v7.1.0)
from dotenv import load_dotenv
load_dotenv()
from umis_rag.agents.explorer import ExplorerRAG

# After (v7.2.0)
from umis_rag.agents.explorer import ExplorerRAG  # ✅ 자동 로드!
```

#### 2. Explorer 검색 결과 활용
```python
# Before
results = explorer.search_patterns("SaaS", top_k=3)
for doc, score in results:
    pattern_id = doc.metadata.get('pattern_id')

# After (권장)
results = explorer.search_patterns("SaaS", top_k=3)
patterns = explorer.get_pattern_details(results)  # ✅ 간편!
for p in patterns:
    print(f"{p['pattern_id']}: {p['score']:.4f}")
```

#### 3. Native Mode 활용
- External LLM 호출 최소화
- RAG 검색 + Cursor LLM 분석
- 비용 절감 + 품질 향상

### 아키텍처

1. **Native Mode가 최적**
   - Cursor LLM > External API
   - 비용 $0, 품질 최고
   - 일회성 분석에 이상적

2. **RAG가 핵심 가치**
   - 객관적 패턴 제공

   - 재현 가능성

   - 헬퍼 메서드 → 사용성 ↑
   - 작은 개선이 큰 효과

   - 모든 계산 수식으로
   - 시트 간 연결 필수

   - RAG + Cursor LLM 조합
