# UMIS v5.1.3 모듈화 버전 - 사용 예제

## 🎯 시나리오: 피부과 시술 시장 분석

### 1단계: 프로젝트 설정

**ChatGPT 프로젝트에 첨부할 파일:**
- `workflows/adaptive_workflow_v5.yaml` (MOwner 역할 포함)
- `agents/manalyst_albert.yaml`
- `agents/mexplorer_steve.yaml`

**참고**: MOwner는 당신(사용자)의 역할을 정의합니다. 에이전트가 아니므로 별도 파일이 없으며, workflow에 포함되어 있습니다.

### 2단계: 초기 대화

```
사용자: 피부과 시술 시장을 분석하고 싶습니다. 현재 명확도는 3/10 정도입니다.

ChatGPT: UMIS v5.1.3를 사용하여 피부과 시술 시장을 분석하겠습니다.

Stage 1: Progressive Narrowing (점진적 구체화)을 시작합니다.

[Albert 활성화]
시장을 어떤 관점에서 보시면 도움이 될까요?

1. 💰 돈의 흐름으로 보기
   - 시술 가격대와 결제 패턴
   - 병원-환자 직거래 vs 플랫폼 중개
   - 보험 적용 vs 비급여 시술

2. 🏢 시장 참여자로 보기
   - 대형 네트워크 vs 개인 클리닉
   - 피부과 vs 성형외과 경계
   - 의료진 vs 관리사 시술

3. 📍 실제 거래 패턴으로 보기
   - 시술 종류별 빈도
   - 재방문율과 패키지 구매
   - 지역별 가격 차이

어느 관점이 가장 관심 있으신가요?
```

### 3단계: 사용자 선택 후

```
사용자: 1번 돈의 흐름으로 보고 싶습니다.

ChatGPT: [Albert의 classification 수행]

돈의 흐름 관점에서 다음과 같이 세분화할 수 있습니다:

거래 주체별:
- B2C: 병원 → 개인 고객
- B2B2C: 병원 → 플랫폼 → 고객
- B2G: 병원 → 보험공단 → 환자

거래 규모별:
- 고가 시술: 100만원 이상 (레이저, 필러 등)
- 중가 시술: 30-100만원 (보톡스, 관리 등)
- 저가 시술: 30만원 미만 (기본 관리)

어떤 측면을 더 깊이 보시겠습니까?
```

### 4단계: Stage 2 - MECE 의도 파악

```
사용자: 플랫폼을 통한 거래에 관심이 있습니다.

ChatGPT: [Stage 2 - Albert의 MECE 분류 활성화]

플랫폼 거래와 관련하여 귀하의 의도를 명확히 파악하겠습니다:

1. 💰 투자/자산 관점
   - 플랫폼 기업 인수/투자
   - 병원 체인 투자
   
2. 🚀 신규 사업 관점
   - 새로운 중개 플랫폼 구축
   - 기존 플랫폼과 차별화된 서비스
   
3. 🔗 기존 사업 연계 관점
   - 병원 운영 중 플랫폼 활용
   - 플랫폼과의 제휴/파트너십

어느 관점에 해당하시나요?
```

### 5단계: Steve의 기회 해석

```
사용자: 2번 신규 사업 관점입니다.

ChatGPT: [Steve 활성화 - Albert의 관찰 기반 해석]

Albert의 관찰을 바탕으로 Steve가 기회를 해석합니다:

🔍 발견된 비효율성:
1. 높은 중개 수수료 (20-30%)
2. 정보 비대칭 (가격/품질 불투명)
3. 복잡한 예약/결제 프로세스

💡 환경 변화 기회:
1. MZ세대의 투명성 요구 증가
2. 의료 마이데이터 정책 변화
3. AI 기반 개인화 추천 기술

추천 깊이:
- Level 1: Quick Facts (10분)
- Level 2: Structured Insights (30분) ← 추천
- Level 3: Comprehensive Report (2시간)

어느 수준으로 분석할까요?
```

### 6단계: 추가 에이전트 활성화

```
사용자: Level 2로 진행하되, 시장 규모도 알고 싶습니다.

ChatGPT: Bill을 추가로 활성화하시겠습니까? 
시장 규모 계산을 위해 mquant_bill.yaml 파일을 첨부해주세요.

[Bill 활성화 후]

Bill이 4가지 방법으로 플랫폼 중개 시장을 계산합니다:

1. Top-down: 전체 피부과 시장 → 플랫폼 비중
2. Bottom-up: 플랫폼 거래수 × 평균 단가
3. Proxy: 강남언니 추정 매출 × 시장점유율
4. Competitor: 주요 플랫폼 매출 합산

예상 SAM: 3,000-4,000억원 (±25%)
```

### 7단계: 검증 필요시

```
사용자: 데이터의 신뢰성을 확인하고 싶습니다.

ChatGPT: Rachel을 활성화하시겠습니까?
mvalidator_rachel.yaml 파일을 첨부해주세요.

[Rachel 활성화 후]
데이터 정의와 신뢰성을 검증합니다...
```

## 💡 핵심 포인트

1. **필요한 만큼만 활성화**: 기본 3개 파일로 시작
2. **명시적 협업**: 에이전트 간 데이터 흐름 명확히
3. **단계별 진행**: 각 Stage의 목적 이해
4. **유연한 깊이**: 상황에 맞는 분석 수준 선택
