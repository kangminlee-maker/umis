## 🎯 추정 목표

**Question**: 국내 중소/스타트업 대상 통합 마케팅 CRM 시장 규모는?

**정의 (Scope)**:
- **지역**: 대한민국
- **제품**: 통합 마케팅 CRM (이메일 + 소셜 + 자동화 + CDP)
- **고객**: 직원 10-100명 중소기업/스타트업
- **거래 모델**: 구독형 SaaS (월/연)

**목표 정확도**: ±50% (자릿수 파악, Discovery 단계)

**공식**:
```
SAM = 타겟 기업 수 × 도입 가능 비율 × 평균 객단가 × 12개월
```

### Step 3: 분해 및 계산

#### 3.1 타겟 기업 수 (중소/스타트업 10-100명)

**출처 1: 프로젝트 데이터** (확정 숫자 없음)

**출처 2: LLM 직접 답변**
- 한국 전체 기업 수: 약 750만 개 (2023년 기준)
- 직원 10-100명 비율: 약 3-5%

**출처 3: 검색 공통 맥락**
- 중소기업 정의: 300명 미만
- 소기업: 50명 미만
- 10-100명: 소기업 내 상위 + 중기업 하위

**계산**:
```
전체 기업: 750만
10-100명 비율: 4% (3-5% 중간값)
타겟 기업 수: 750만 × 4% = 30만 개
```

**Boundary 검증**:
- 상한: 50명 이상 기업 약 10% → 750만 × 10% = 75만 개 ✓ (30만 < 75만)
- 하한: 100명 이상 기업 약 1% → 750만 × 1% = 7.5만 개 ✓ (30만 > 7.5만)

**추정치 1**: **30만 개 ± 15%** (25.5만 ~ 34.5만)

---

#### 3.2 산업 필터 (마케팅 니즈 높은 산업)

**포함 산업**:
- B2C 커머스 (쇼핑몰, D2C)
- B2B SaaS
- 서비스업 (학원, 병원, 뷰티)
- 콘텐츠/미디어

**제외 산업**:
- 제조업 (마케팅 < 영업)
- 건설/부동산
- 전통 도소매

**비율 추정** (출처 7: Rule of Thumb):
- 일반적으로 B2C + 서비스업 = 전체의 30-40%
- 디지털 마케팅 니즈: 추가 20% (B2B SaaS, 콘텐츠)

**계산**:
```
마케팅 니즈 산업 비율: 35% (보수적)
타겟 가능 기업: 30만 × 35% = 10.5만 개
```

**추정치 2**: **10.5만 개 ± 20%** (8.4만 ~ 12.6만)

---

#### 3.3 도입 가능 비율 (Addressable Rate)

**현황 관찰** (Albert 관찰):
- 현재 사용 중: 추정 10-15% (스티비 20만 사용자 중 일부)
- 미도입 but 니즈 있음: 25-30%
- 전혀 니즈 없음: 55-65%

**도입 가능 비율**: 현재 사용 + 니즈 있음
```
도입 가능: 15% (현재) + 25% (잠재) = 40%
```

**Behavioral Economics 보정** (출처 5):
- Status Quo Bias: 현상 유지 선호 → 실제 전환율 ×0.5
- 최종 도입 가능: 40% × 0.5 = **20%** (보수적)

**계산**:
```
실제 타겟 시장: 10.5만 × 20% = 2.1만 개
```

**추정치 3**: **2.1만 개 ± 30%** (1.5만 ~ 2.7만)

---

#### 3.4 평균 객단가 (ARPU)

**시장 조사** (출처 3: 검색 공통 맥락):
- 국내 CRM: 월 5만~30만원
- 글로벌 CRM: 월 $50~$500 (6.5만~65만원)

**타겟 세그먼트 (중소/스타트업) 가격대**:
- 저가 (Freemium 유료): 월 5만원
- 중가 (Standard): 월 10-15만원
- 고가 (Pro): 월 20-30만원

**비율 추정** (출처 6: 통계 패턴 - Power Law 80-20):
- 저가 50% × 5만 = 2.5만
- 중가 40% × 12만 = 4.8만
- 고가 10% × 25만 = 2.5만

**가중 평균**:
```
ARPU = (2.5만 + 4.8만 + 2.5만) / 3 = 9.8만원
≈ 10만원/월
```

**Benchmark 검증** (출처 7: Rule of Thumb):
- SaaS B2B 중소기업 ARPU: $50-200 (6.5만~26만원)
- 10만원은 범위 내 ✓

**추정치 4**: **월 10만원 ± 20%** (8만 ~ 12만원)

---

### Step 4: 최종 SAM 계산

```
SAM = 타겟 기업 수 × 평균 객단가 × 12개월

타겟 기업: 2.1만 개
ARPU: 월 10만원
연간: 12개월

SAM = 2.1만 × 10만 × 12
    = 2,520억원
    ≈ 2,500억원
```

---

### Step 5: Boundary 검증 (물리적 제약)

**상한 체크**:
- 전체 CRM 시장 (영업 + 마케팅): 약 5,000억~1조원 추정
- 마케팅 CRM = 30-40% → 1,500억~4,000억원
- **2,500억원**은 범위 내 ✓

**하한 체크**:
- 스티비 매출 (추정): 20만 사용자 × 5만원/월 × 12 = 1,200억원
- 스티비는 이메일 only → 통합 시장은 더 큼
- **2,500억원**은 합리적 ✓

**시간 제약**:
- 12개월 = 365일 (고정) ✓

---

### Step 6: 신뢰도 평가

**오차 범위 계산**:
- 타겟 기업 수: ±20% (2.1만 × 0.2 = 0.42만)
- 도입 가능 비율: ±30% (20% × 0.3 = 6%p)
- ARPU: ±20% (10만 × 0.2 = 2만)

**총 오차** (곱셈 오차 전파):
```
Total Error = √(20%² + 30%² + 20%²)
            = √(0.04 + 0.09 + 0.04)
            = √0.17
            ≈ 41%
```

**최종 범위**:
```
SAM = 2,500억원 ± 41%
    = 1,475억 ~ 3,525억원
```

### 추가 지표 (Quick Estimate)

#### TAM (Total Addressable Market)
```
전체 마케팅 CRM 시장 (모든 규모)
= SAM / (타겟 비율)
= 2,500억 / 35% (10-100명 비율)
= 7,143억원
≈ 7,000억원
```

#### SOM (Serviceable Obtainable Market) - 3년 목표
```
현실적 점유율: 5-10% (신규 진입)
SOM = 2,500억 × 5% = 125억원 (보수적)
SOM = 2,500억 × 10% = 250억원 (공격적)

→ 3년 목표: 125억 ~ 250억원
```
