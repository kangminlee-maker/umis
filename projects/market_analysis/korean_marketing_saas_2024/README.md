## 🔍 분석 방법론 (UMIS Native Mode)

### UMIS 5-Agent 시스템

```
Observer → Explorer → Quantifier → Validator → 종합
(구조)   (기회)     (규모)       (검증)     (리포트)
```

#### Agent별 산출물

| Agent | 역할 | 주요 도구 | 산출물 |
|-------|------|----------|--------|
| **Observer** | 시장 구조 분석 | 13차원 정의, 가치사슬 | 01, 02 문서 |
| **Explorer** | 기회 발굴 | RAG 패턴 매칭 | 03 문서 |
| **Quantifier** | 시장 규모 | SAM 4가지 방법 | 04 문서 |
| **Validator** | 데이터 검증 | 출처 확인, Gap 분석 | 05 문서 |
| **Guardian** | 품질 평가 | 종합 검토 | 00 문서 |

---

### Native Mode 특징

**방법론**:
```
1. RAG 패턴 검색 (Python)
   → ExplorerRAG.search_patterns()
   → 5개 비즈니스 모델 패턴 발견

2. 분석 (Cursor Native LLM)
   → Claude Sonnet 4.5가 직접 분석
   → RAG 패턴 + 시장 지식 결합
   → 고품질 산출물 생성

3. 산출물 작성
   → Markdown 문서 6개
   → 추적 가능한 ID 체계 (OBS_*, OPP_*, SAM_*)
```

**장점**:
- ✅ 최고 품질 (Sonnet 4.5)
- ✅ 추가 비용 없음 (API 호출 $0)
- ✅ RAG 패턴 활용 (객관성)
- ✅ 큰 컨텍스트 (200K tokens)
