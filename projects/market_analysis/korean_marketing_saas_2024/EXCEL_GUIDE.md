## 📁 시트 구조 (12개)

### 🎯 핵심 시트 (반드시 확인)

#### 1. **시장규모_요약** (시작점!)
**역할**: 4가지 방법 통합 및 수렴 분석

**주요 셀:**
- `B4~B7`: 4가지 방법 결과 (다른 시트에서 자동 참조)
  - `B4 = M1_TopDown_계산!C14`
  - `B5 = M2_BottomUp_계산!G24`
  - `B6 = M3_Proxy_계산!C34`
  - `B7 = M4_Competitor_계산!E31`
- `B10`: 평균 = `AVERAGE(B4:B7)`
- `B13`: CV = `표준편차 / 평균`
- `B26`: **최종 채택값 2,700억원** (노란색)

**사용법:**
1. 각 방법 시트(M1~M4)에서 가정 변경
2. 이 시트에서 자동 업데이트 확인
3. CV < 30% 확인 (신뢰성 검증)

### 📐 계산 시트 (4가지 방법 상세)

#### 2. **M1_TopDown_계산**
**Method 1: Top-Down (하향식)**

**계산 로직:**
```
글로벌 MA 시장 ($7.53B)
  → 원화 환율 (₩1,330)
  → 글로벌 시장 원화 (B6 = B4 × B5 × 100)
  → 한국 비중:
     A. GDP 비례 (1.7%)
     B. 광고비 비례 (1.71%)
     C. SaaS 성숙도 조정 (1.7% × 1.2)
  → 최종: C14 = C12
```

**주요 셀:**
- `B4`: 글로벌 시장 $7.53B (수정 가능)
- `B5`: 환율 ₩1,330 (수정 가능)
- `C14`: **최종 결과 ~2,040억원**

---

#### 3. **M2_BottomUp_계산**
**Method 2: Bottom-Up (상향식)**

**계산 로직:**
```
각 세그먼트별:
  도입 기업 (D) = 기업 수 (B) × 도입률 (C)
  월 매출 (F) = 도입 기업 (D) × ARPU (E) / 10000
  연 매출 (G) = 월 매출 (F) × 12

총 합계 (G20) = B2B + B2C + 에이전시
중복 제거 (G21) = G20 × 0.85
보수 조정 (G22) = G21 × 0.95
최종 (G24) = ROUND(G22, -1)
```

**주요 셀:**
- `B4~B13`: 각 세그먼트 기업 수 (수정 가능)
- `C4~C13`: 도입률 (ASM_001, ASM_002 등)
- `E4~E13`: ARPU (수정 가능)
- `G24`: **최종 결과 ~3,540억원**

**가정 참조:**
- `C4` (70%): ASM_001 - 대기업 도입률
- `C6` (20%): ASM_002 - SMB 도입률
- `C10` (5%): ASM_006 - 오프라인 사용률

---

#### 4. **M3_Proxy_계산**
**Method 3: Proxy (유사 시장 유추)**

**계산 로직:**
```
일본 시장 (¥480B)
  → 원화 환율 (₩11.25/JPY)
  → 일본 시장 원화 (B6 = B4 × B5 × 10)
  → 한국/일본 비율:
     - GDP: 43%
     - 광고비: 48%
     - 기업 수: 42%
  → 가중 평균 (C21)
  → SaaS 성숙도 조정 (C26 = C21 × 85%)
  → 성장 격차 (C32 = C26 × 1.4)
  → 최종: C34 = ROUND(C32, -1)
```

**주요 셀:**
- `B4`: 일본 시장 ¥480B
- `D10~D14`: 한국/일본 비율 (수정 가능)
- `C29~C30`: 성장 격차 배율
- `C34`: **최종 결과 ~2,760억원**

---

#### 5. **M4_Competitor_계산**
**Method 4: Competitor Revenue (경쟁사 매출 역산)**

**계산 로직:**
```
각 플레이어:
  연 매출 (E) = 고객 수 (C) × ARPU (D) × 12 / 10000

Tier별 소계:
  Tier 1 (E19) = HubSpot + Salesforce
  Tier 2 (E20) = 7개 국내 기업
  Tier 3 (E21) = 50개 스타트업

총 매출 (E23) = E19 + E20 + E21
시장 규모 (E28) = E23 / 점유율 (55%)
조정 (E29) = E28 × 1.1
최종 (E31) = ROUND(E29, -2)
```

**주요 셀:**
- `C6~C17`: 각 플레이어 고객 수 (수정 가능)
- `D6~D17`: ARPU (수정 가능)
- `E27`: 점유율 55% (ASM_004)
- `E31`: **최종 결과 ~2,400억원**

## 🎯 핵심 계산식

### Method 1: Top-Down

```excel
[M1_TopDown_계산]

글로벌 시장 (B6) = 글로벌 MA ($7.53B) × 환율 (₩1,330) × 100

한국 비중:
  GDP 비례 (C10) = B6 × (1.79 / 105)
  광고비 비례 (C11) = B6 × (12 / 700)
  SaaS 성숙도 (C12) = C10 × 1.2

최종 (C14) = C12
```

### Method 2: Bottom-Up

```excel
[M2_BottomUp_계산]

각 세그먼트:
  도입 기업 (D) = 기업 수 (B) × 도입률 (C)
  월 매출 (F) = 도입 기업 (D) × ARPU (E) / 10000
  연 매출 (G) = 월 매출 (F) × 12

총 합계 (G20) = SUM(B2B + B2C + 에이전시)
중복 제거 (G21) = G20 × 0.85
보수 조정 (G22) = G21 × 0.95
최종 (G24) = ROUND(G22, -1)
```

### Method 3: Proxy

```excel
[M3_Proxy_계산]

일본 시장 (B6) = ¥480B × ₩11.25/JPY × 10

가중 평균 (C21) = GDP (C18) × 0.3 + 광고비 (C19) × 0.4 + 기업 (C20) × 0.3
성숙도 조정 (C26) = C21 × 85%
성장 배율 (C31) = 1.25 × 1.15
최종 (C34) = ROUND(C26 × C31, -1)
```

### Method 4: Competitor Revenue

```excel

  연 매출 (E) = 고객 (C) × ARPU (D) × 12 / 10000

  Tier 1 (E19) = SUM(HubSpot, Salesforce)
  Tier 2 (E20) = SUM(7개 국내)
  Tier 3 (E21) = 50개 스타트업

시장 규모 (E28) = E23 / 55%
```

## 🐛 문제 해결

### 문제: #REF! 에러

**원인**: 시트 간 참조 깨짐  
**해결**: 시트 이름 확인 (M1_TopDown_계산, M2_BottomUp_계산 등)

### 문제: 값이 0 또는 이상함

**원인**: 순환 참조 또는 수식 오류  
**해결**: Ctrl+` (수식 표시) 후 수식 확인

### 문제: CV가 너무 높음 (>30%)

**원인**: 4가지 방법 간 편차 큼  
**해결**: 
1. 각 방법의 가정 재검토
2. 이상값 있는지 확인
3. 데이터 출처 재확인
