## 1. 브랜드 개요

### 1.1 가벼운학습지 (MYLIGHT)

#### 1.1.1 기본 정보
- **브랜드명**: 가벼운학습지
- **모기업**: MYLIGHT  
- **웹사이트**: mylight.co.kr
- **주요 사업**: 패키지형 학습 상품 제작 및 판매
- **설립 시기**: 2020년대 초반 (추정)

#### 1.1.2 브랜드 현황
- **사업 영역**: 외국어 학습지에서 시작하여 취미, 부업, 데뷔 과정으로 확장
- **주요 특징**: 학습지 + 관련 도구 + 학습 영상을 패키지로 제공
- **차별화 요소**: IP 콜라보레이션 (짱구, 토이스토리 등)

### 1.2 뉴스프레소 (Newspresso)

#### 1.2.1 기본 정보
- **브랜드명**: 뉴스프레소 (Newspresso)
- **웹사이트**: newspresso.kr
- **주요 사업**: 비즈니스 영어 교육 콘텐츠 제작 및 서비스
- **설립 시기**: 2020년대 초반 (추정)

#### 1.2.2 브랜드 현황
- **사업 영역**: 외국 시사잡지 콘텐츠 라이센싱 기반 교육
- **주요 특징**: 프리미엄 시사 콘텐츠 + 한국어 해석 + 학습 지원
- **차별화 요소**: 글로벌 권위 있는 매체와의 라이센싱 계약

## 2. 제품 및 서비스 현황

### 2.1 가벼운학습지 제품 라인업

#### 2.1.1 외국어 학습지 (현재 운영 중)
```
제공 언어: 5개
├── 영어
├── 일본어  
├── 중국어
├── 스페인어
└── 프랑스어

학습 방식:
├── 커리큘럼: 하루 15분, 주 1권 완성
├── 구성: 학습지 + 학습 영상
└── 특별 버전: IP 콜라보 (짱구, 토이스토리)
```

#### 2.1.2 취미 학습지 (현재 운영 중)
```
주요 분야:
├── 악기 연주 (악기 + 학습지 + 영상 패키지)
├── 노래 부르기 ("잠골버스의 방구석 노래방 데뷔")
├── 기타 창작 활동
└── 패키지 구성: 물리적 도구 + 학습 콘텐츠 통합 제공
```

#### 2.1.3 부업/데뷔 과정 (현재 운영 중)
```
제공 과정:
├── 문구작가 (재미 + 부업 결합형)
├── 작사가 데뷔 (실제 음원 플랫폼 크레딧 제공)
├── 기타 창작 분야
└── 특징: 학습 + 실제 커리어 기회 연결
```

### 2.2 뉴스프레소 서비스 라인업

#### 2.2.1 프리미엄 에디션 (현재 운영 중)
```
라이센싱 파트너:
├── 월스트리트 저널 (금융/비즈니스)
├── 하버드 비즈니스 리뷰 (경영 인사이트)
├── 타임 (글로벌 시사)

```

#### 2.2.2 서비스 구성 (현재 제공 중)
```
기본 서비스:
├── 기사 원문 + 한국어 번역/해석
├── 정치/사회/문화적 맥락 설명
├── MP3 음원 제공
└── 비즈니스 영어 표현 학습

프리미엄 서비스:
├── 1:1 영작 첨삭
├── 개인 맞춤 학습 진도 관리
├── 실무 적용 가이드
└── B2B 기업 교육 서비스
```
