# Personal Layer

**우선순위:** 1 (최우선)  
**접근:** Owner only  
**검증:** None (자유 실험)

---

## 목적

개인 실험 및 Draft 데이터

---

## 구조

```
personal/
└── {user_name}/
    ├── experiments.yaml
    ├── draft_ideas.yaml
    └── scratch.yaml
```

---

## 특징

```yaml
자유:
  • 검증 불필요
  • 실험 가능
  • 격리됨

검색:
  • 최우선 (Personal > Team > Core)
  • 자신의 실험이 공식 데이터보다 우선

승격:
  • 성공 시 Team으로 승격 가능
  • Team Lead 승인 필요
```
