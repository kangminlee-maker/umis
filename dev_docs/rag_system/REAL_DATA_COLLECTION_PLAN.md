### Tier S: 공식 통계 (신뢰도 100%)

**1. 통계청 기업경영분석** ⭐⭐⭐
```yaml
URL: https://kostat.go.kr/menu.es?mid=a30103000000
접근: 웹사이트 → 통계표 → 산업별 재무비율
커버: 전 산업 (제조, 도소매, 서비스업)
샘플: 50,000개+ 기업
업데이트: 연 1회

사용 방법:
  1. 통계청 웹사이트 접속
  2. "기업경영분석" 검색
  3. "산업별 재무비율" 선택
  4. KSIC 산업분류별 데이터 다운로드

  - 음식점업 (KSIC 56): 영업이익률 8.9%
  - 소매업 (KSIC 47): 영업이익률 3.2%

```

## 📋 추정 로직 템플릿

### 예시: 헬스장
```yaml
benchmark_id: "margin_korea_fitness_001"
industry: "헬스장"
region: "한국"

margins:
  operating_margin: {median: 0.18}

data_source:
  primary: "ESTIMATED"
  
  estimation_logic: |
    ## 추정 방법
    
    ### Step 1: 기본 가정
    - 회원 수: 200명 (중형 헬스장)
    - 월 회원비: 70,000원
    - 유지율: 70% (업계 평균)
    - 월 매출: 200명 × 70,000원 × 70% = 9,800,000원
    
    ### Step 2: 비용 구조
    - 인건비: 35% (트레이너 4명 × 250만원 = 1,000만원)
      → 월 매출 대비: 9.8M / 3.4M = 35%
    - 임대료: 25% (300평 × 30,000원/평 = 900만원 보증금 전환)
      → 2.5M/월
    - 기구 감가상각: 8% (초기 투자 2억 / 5년)
      → 약 3.3M/월
    - 공과금: 6% (전기, 수도)
    - 기타: 8%
    
    ### Step 3: 영업이익 계산
    매출: 9.8M
    비용: 8.0M (35%+25%+8%+6%+8% = 82%)
    영업이익: 1.8M
    영업이익률: 18%
    
    ### Step 4: 교차 검증
    - 통계청 "스포츠 서비스업" 참조: OPM 15-20%
    - 프랜차이즈 정보공개서 (애니타임피트니스 등): OPM 15-22%
    - 일치 확인
  
  assumptions:
    - "회원 200명 (중형)"
    - "월 회원비 7만원"
    - "유지율 70%"
    - "트레이너 4명"
    - "300평 공간"
  
  confidence_range: "±5-8%p"
  
  cross_check:
    - source: "통계청 스포츠 서비스업"
      result: "15-20% (일치)"
    - source: "애니타임피트니스 정보공개서"
      result: "예상 OPM 15-22% (일치)"
  
  collection_date: "2025-11-12"

reliability: "estimated_transparent"
estimation_quality: "good"
```
