# DART Crawler 개발 문서

DART (Data Analysis, Retrieval and Transfer System) 크롤러 관련 개발 문서입니다.

## 주요 문서

- `DART_CRAWLER_QUICKSTART.md`: 빠른 시작 가이드
- `DART_CRAWLER_DESIGN.md`: 시스템 설계 문서
- `DART_CRAWLER_IMPLEMENTATION_SUMMARY.md`: 구현 요약
- `DART_API_LIMITATION_ANALYSIS.md`: API 제한사항 분석
- `DART_SGA_COMPLETE.md`: SGA 크롤링 완료 보고서

## 주요 기능

- DART Open API 연동
- 기업 재무제표 크롤링
- SGA (판매비와관리비) 데이터 파싱
- Rate Limiting 및 에러 핸들링

- `umis_rag/data_sources/dart/`
