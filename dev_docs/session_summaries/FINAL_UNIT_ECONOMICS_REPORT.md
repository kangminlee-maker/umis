## 💡 핵심 통찰

### 1. 공헌이익 vs 영업이익

**공헌이익(CM) = Unit Economics:**
- 비즈니스 본질
- 확장성 지표
- **영업이익보다 중요!**

**차이 = 고정비:**
```
아모레퍼시픽: CM 24.6% - OM 3.4% = 21.2%p (고정비)
유한양행:     CM 21.6% - OM 3.2% = 18.4%p
GS리테일:     CM 14.4% - OM -0.5% = 14.9%p
LG전자:      CM 14.2% - OM 2.0% = 12.2%p
이마트:       CM 8.5% - OM 1.2% = 7.3%p
BGF리테일:    CM 7.9% - OM 3.0% = 4.9%p (효율!)
CJ ENM:      CM 5.0% - OM 6.0% = -1.0%p (고정비 낮음)
```

### 2. 비즈니스 모델별

**가맹점 모델 (GS, BGF):**
- 변동비: 매우 높음 (수수료, 임대)
- 고정비: 상대적 낮음
- 확장성: 높음
- CM: 7.9-14.4%

**직영 모델 (이마트):**
- 변동비: 중간
- 고정비: 높음
- 규모의 경제: 필수
- CM: 8.5%

**제조 (고마진):**
- GM: 높음 (28-65%)
- 브랜드/기술 투자
- CM: 14-25%

**엔터:**
- GM: 높음 (46%)
- 컨텐츠 비용 (변동)
- CM: 5%
