## 🚀 즉시 사용 가능

### 완전한 워크플로우
```python
# 1. 시장 분석 (Observer)
observer = get_observer_rag()
market_report = observer.analyze_market(...)

# 2. 시계열 분석 (Observer - Gap #1)
timeline = observer.analyze_market_timeline(
    market="음악 스트리밍",
    start_year=2015,
    end_year=2025
)
# → 변곡점, 진화 패턴, Deliverable 자동 생성

# 3. 기회 발굴 (Explorer)
explorer = create_explorer_agent()
opportunity = explorer.generate_opportunity_hypothesis(...)

# 4. SAM 계산 (Quantifier)
quantifier = get_quantifier_rag()
sam = quantifier.calculate_sam(...)

# 5. 경쟁사 마진 추정 (Estimator - Gap #2)
estimator = get_estimator_rag()
competitor_margin = estimator.estimate(
    question="경쟁사 영업이익률은?",
    project_data={
        'industry': 'SaaS',
        'sub_category': 'B2B Enterprise',
        'company_size': 'scale'
    }
)
# → ±12% 오차로 정확!

# 6. 실행 전략 (Explorer - Gap #3)
playbook = explorer.generate_strategy_playbook(
    validated_opportunity=opportunity,
    market_context=market_report,
    quantified_market=sam
)
# → GTM, Roadmap, Milestone, Excel 자동!

# 완전한 파이프라인! 🎉
```
