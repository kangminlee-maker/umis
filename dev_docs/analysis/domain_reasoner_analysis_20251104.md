- **차원 분석**:
  - 수수료율(%) = 플랫폼 수수료 / 거래액 × 100
  - 단위 일관성: KRW / KRW × 100 = % ✓
- **보존 법칙**:
  - 거래액 = 음식가 + 배달비 + 플랫폼 수수료
  - 음식점 실수령 = 거래액 × (1 - 수수료율 - 배달비율)
- **비례 관계**:
  - 수수료율 ↑ → 음식점 수익 ↓ → 이탈 위험 ↑
  - 수수료율 ↓ → 플랫폼 수익 ↓ → 서비스 품질 ↓
- **판단**: 수학적 관계 명확, 균형점 존재

  - **분자**: 플랫폼 수수료만 (배달비 제외)
