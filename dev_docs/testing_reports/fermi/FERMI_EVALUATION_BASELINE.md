## 문제 1: 한국 전체 사업자 수

### 실제 데이터 (정답)
**7,837,000개** (2023년 통계청, 국세청 사업자등록 기준)

---

### AI의 Fermi 분해 (기준선)

#### 접근 방법: Bottom-up

```
목표: 한국 전체 사업자 수 추정

분해 단계:

1단계: 한국 인구 및 경제활동인구
- 한국 인구: 5,200만명
- 경제활동인구 (15-64세): 60% = 3,120만명
- 실제 경제활동참가율: 65% = 2,030만명

2단계: 사업자 비율 추정
- 근로자 (고용): 80% = 1,624만명
- 자영업자/사업자: 20% = 406만명

3단계: 법인 사업자 추가
- 자영업자: 406만개 (1인당 1개)
- 중소/중견/대기업: 법인사업자
  - 근로자 1,624만명 / 평균 20명 = 81만개
- 법인 추가 보정 (다중 사업장 등): ×1.2 = 97만개

4단계: 총 사업자 수
- 자영업: 406만개
- 법인: 97만개
- 휴업/기타: ×1.2 보정 = 603만개
- 합계: 약 600만개

최종 추정: 600만개 (실제 784만개, 오차 23%)
```

#### AI 분해 분석

**강점:**
- 명확한 3단계 분해
- 인구 기반 bottom-up
- 합리적인 가정 (경제활동인구 20%)

**약점:**
- 실제보다 낮게 추정 (600만 vs 784만)
- 휴업 사업자, 부업 등 고려 부족
- 1인 자영업자의 다중 사업자등록 미반영

**보완한 분해:**

```
개선된 분해:

1단계: 경제활동인구
- 2,030만명 (동일)

2단계: 사업자 참여율 (더 높게)
- 자영업자: 25% = 508만명
- 각자 평균 1.3개 사업자등록 = 660만개

3단계: 법인 사업자
- 법인: 120만개 (보정 후)

4단계: 합계
- 660만 + 120만 = 780만개

최종 추정: 780만개 (실제 784만개, 오차 0.5%)
```

## 문제 2: 서울시 인구

### 실제 데이터 (정답)
**9,668,465명** (2024년 1월 기준, 서울시 통계)

---

### AI의 Fermi 분해 (기준선)

#### 접근 방법: Top-down

```
목표: 서울시 인구 추정

분해 단계:

1단계: 한국 전체 인구
- 한국 인구: 5,200만명

2단계: 서울 비중 추정
- 수도권 집중도: 50% = 2,600만명
- 서울:경기:인천 비율 = 1:2:0.5
- 서울 비중: 1 / 3.5 = 28.6%
- 서울 인구: 2,600만 × 0.286 = 744만명

대안 접근: 서울 면적 기반
- 한국 면적: 10만 km²
- 서울 면적: 605 km² (0.6%)
- 도시 밀집도 보정: ×300배
- 서울 인구: 5,200만 × 0.006 × 300 = 936만명

최종 추정: 800~940만명 (실제 967만명, 오차 3%)
```

#### AI 분해 분석

**강점:**
- 두 가지 방법 제시 (비율 + 면적)
- 범위 추정으로 불확실성 표현
- 합리적 추정 (오차 3%)

**약점:**
- 수도권 비율 가정(50%)의 근거 부족
- 서울:경기:인천 비율 단순화

## 문제 3: 한국 커피 전문점 수

### 실제 데이터 (정답)
**약 100,000개** (2023년, 스타벅스 1,800개 + 기타 추정)

---

### AI의 Fermi 분해 (기준선)

#### 접근 방법: Bottom-up

```
목표: 한국 커피 전문점 수 추정

분해 단계:

1단계: 잠재 고객층
- 한국 인구: 5,200만명
- 커피 소비층 (20-60세): 50% = 2,600만명
- 정기 소비자 (주 3회 이상): 40% = 1,040만명

2단계: 점포당 고객 수
- 한 점포 상권: 반경 500m
- 상권 인구: 도시 3,000명, 시골 1,000명
- 평균 인구: 2,000명
- 정기 소비자 비율 40%: 800명/점포

3단계: 총 점포 수
- 전국 정기 소비자: 1,040만명
- 점포당 고객: 800명
- 필요 점포: 1,040만 / 800 = 13,000개

4단계: 경쟁 보정
- 브랜드 다양성 (스타벅스, 이디야, 개인카페 등)
- 중복 상권: ×6배
- 총 점포: 13,000 × 6 = 78,000개

최종 추정: 78,000개 (실제 100,000개, 오차 22%)
```

#### AI 분해 분석

**강점:**
- 고객 중심 bottom-up
- 상권 개념 도입
- 경쟁 보정 고려

**약점:**
- 중복 상권 계수(6배) 근거 부족
- 실제보다 낮게 추정

## 평가 기준

### 1. 정확도 (40점)

**AI 기준 대비 오차:**
```python
score = 40 × (1 - |model_value - true_value| / true_value)
```

- 오차 0%: 40점
- 오차 10%: 36점
- 오차 50%: 20점
- 오차 100%: 0점

### 2. 분해 합리성 (30점)

**AI 분해와 비교:**

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 단계 수 | 5점 | AI와 유사한 3-4단계 |
| 가정의 합리성 | 10점 | AI 수준 이상의 근거 |
| 수치의 보편성 | 10점 | 누구나 동의 가능한 숫자 |
| 계산 논리 | 5점 | 각 단계 연결 명확 |

### 3. 목표 정의 근접성 (30점)

**분해가 목표를 잘 추정하는가?**

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 접근 방법 적절성 | 10점 | top-down/bottom-up 적합 |
| 핵심 변수 포함 | 10점 | 목표에 직접 영향 주는 변수 |
| 최종 계산 일관성 | 10점 | 각 단계가 최종값으로 이어짐 |

## 모델 평가 예시

### 모델 A의 분해 (사업자 수 문제)

```
1단계: 인구 5,000만
2단계: 경제활동 20% = 1,000만
3단계: 사업자 비율 10% = 100만개
최종: 100만개
```

**평가:**
- 정확도: 100만 vs 784만 (오차 87%) → 5점
- 합리성: 단계 부족, 법인 미고려 → 10점
- 목표 근접성: 너무 단순 → 10점
- **총점: 25/100점**

### 모델 B의 분해

```
AI와 유사한 수준의 분해
최종: 750만개
```

**평가:**
- 정확도: 750만 vs 784만 (오차 4%) → 38점
- 합리성: AI 수준 → 25점
- 목표 근접성: 적절 → 25점
- **총점: 88/100점**
