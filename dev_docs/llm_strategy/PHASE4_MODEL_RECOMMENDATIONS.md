## ⚠️ 주요 위험 요소

### 1. Phase 4 미검증 문제

```yaml
문제:
  - o3-mini, o4-mini는 Phase 0만 테스트됨
  - Phase 0 ≠ Phase 4 (완전히 다른 작업)
  - 실제 Phase 4 성능 알 수 없음

해결:
  ✅ 즉시 Phase 4 테스트 필요!
  - o3-mini-2025-01-31 (최우선)
  - o4-mini-2025-04-16
  - o3-2025-04-16
```

### 2. 토큰 사용량 불확실성

```yaml
Phase 0 토큰: 100-300개
Phase 4 토큰: 1,000-2,000개 (10-20배!)

영향:
  - 추정 비용이 10-20배 증가 가능
  - o3-mini: $0.87 → $8.70/1,000회
  - o4-mini: $0.60 → $6.00/1,000회

대응:
  실제 Phase 4 테스트로 확인 필요
```

### 3. 품질 vs 비용 트레이드오프

```yaml
고품질 (검증됨):
  - o1-2024-12-17: $22.54/1,000회, 90/100 ✅
  
저비용 (미검증):
  - o3-mini: $0.87-$8.70/1,000회, ??/100 ❓
  - o4-mini: $0.60-$6.00/1,000회, ??/100 ❓

위험:
  저비용 모델이 실제로는 품질 낮을 가능성
  → Phase 4 테스트 필수!
```
