## 🔄 보고서 타입

### 1. 세션 보고서 (Session Reports)
**목적**: 일일 작업 내용 기록  
**형식**: TODAY_*, SESSION_*  
**빈도**: 매일 또는 세션 종료 시

### 2. 완료 보고서 (Completion Reports)
**목적**: 주요 기능 완성 기록  
**형식**: *_COMPLETE, COMPLETION_*  
**빈도**: 기능 완성 시

### 3. 배포 보고서 (Deployment Reports)
**목적**: 배포 및 통합 기록  
**형식**: DEPLOYMENT_*, RELEASE_*  
**빈도**: 배포 시

### 4. 정리 보고서 (Organization Reports)
**목적**: 구조 정리 및 리팩토링 기록  
**형식**: CLEANUP_*, ORGANIZATION_*  
**빈도**: 정리 작업 시
